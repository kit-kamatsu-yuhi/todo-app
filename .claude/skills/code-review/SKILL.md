# コードレビュー Skill

AI生成コードの品質ゲート。基本チェックリスト・PR サイズ制限は `code-review` rule 参照。

## レビュー観点（rule に追加する視点）

### パフォーマンス

- N+1 クエリが発生していないか
- 不要なループ・再計算がないか
- 適切なデータ構造が選択されているか（`data-structures-algorithms` skill 参照）

### セキュリティ

- `security` skill の観点に従う

## レビューコメントの書き方

- 具体的な改善案を示す
- 「なぜ」問題なのかを説明する
- 重要度を明示する（Must / Should / Nit）
