# データ構造とアルゴリズム Skill

適切なデータ構造選択とアルゴリズム設計の指針。計算量表や一般的なアルゴリズムの知識は省略する。

## プロジェクト固有の選定指針

- `code-quality` rule の複雑度基準（循環的複雑度10以下）を満たすアルゴリズムを選択する
- 時間計算量と空間計算量の両方を考慮し、トレードオフを明示する
- N+1 クエリなどの実用的なパフォーマンス問題を優先的に解決する
