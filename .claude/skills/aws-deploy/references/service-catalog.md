# AWS サービスカタログ

## コンピュート

| サービス | 課金モデル | 最小構成月額目安 | ユースケース |
|---------|-----------|----------------|------------|
| App Runner | vCPU + メモリ × 稼働時間 | $5〜 | Web API、プロトタイプ |
| ECS Fargate | vCPU + メモリ × 稼働時間 | $15〜 | 本番ワークロード |
| Lambda | リクエスト数 + 実行時間 | $0〜（無料枠内） | イベント駆動、API |
| Amplify Hosting | ビルド時間 + ホスティング | $0〜（無料枠内） | SPA + API |
